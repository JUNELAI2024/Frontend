Formatting United States Telephone Numbers in Javascript

Format local phones using regular expression:

Unformatted:
1234567890
Formatted:
Javascript Formatting Code:
{FORMATTED-NUMBER} = {UNFORMATTED-NUMBER}.replace(/(\d{3})(\d{3})(\d+)/g, "($1) $2-$3");

Source Code

"1234567890".replace(/(\d{3})(\d{3})(\d+)/g, "($1) $2-$3");
// ... => (123) 456-7890

// Format long distance phones using regular expression.
// Assumes 1 is hardcoded and not stored with value:
"1234567890".replace(/(\d{3})(\d{3})(\d+)/g, "1-$1-$2-$3");

// Format long distance phones using regular expression.
// Assumes 1 is stored with value:
"11234567890".replace(/(\d{1})(\d{3})(\d{3})(\d+)/g, "$1-$2-$3-$4");
